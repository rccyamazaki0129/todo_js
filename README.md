# MyToDo
TODO desktop application using Electron.

## How to install
### install Electron
```npm install --save-dev electron```

## How to start
```npm start```

## How to build
 - Mac
    - ```npm run make```
 - Windows
    - ```npx electron-builder --win --x64```
    - ```npm run package-win```
